# Razorpay Firebase Extensions

This repository contains Firebase Extensions built for [Razorpay](https://razorpay.com/) payment integration.

Each directory contains the source code for an extension and a README explaining how it works.

## Extension Directory

| Extension | Description |
|---|---|
| [firestore-razorpay-payments](firestore-razorpay-payments/) | Sync Razorpay customers, orders, subscriptions, and webhooks with Cloud Firestore |

## Sample App

The [`sample-app/`](sample-app/) directory contains a Next.js application that demonstrates installing and using the extension with Firebase Emulators.

## Contributing

See the individual extension README for setup and testing instructions.
